# nayepankh-foundation-website
Nayepankh Foundation Website – A modern responsive NGO website promoting education, volunteering, and community impact featuring dark mode, impact statistics, gallery, testimonials, and donation sections.

## Features
- Responsive Design
- Dark Mode Toggle
- About Us Section
- Programs & Initiatives
- Impact Statistics
- Event Timeline
- Image Gallery
- Testimonials
- Donation Section
- Volunteer Registration Form
- Contact & Social Media Links

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
To create a professional online presence for Nayepankh Foundation and showcase its mission, activities, and impact on society.
